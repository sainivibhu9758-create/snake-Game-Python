# 🐍 Nokia Retro Snake Game

A retro-style Snake Game built with Python and Tkinter, inspired by the classic Nokia Snake.

## ✨ Features
- 🎮 Classic Snake Gameplay
- 🏆 High Score Saving
- 💻 Modern Nokia-inspired UI
- ⌨️ Arrow Keys / WASD Controls
- 🔄 Restart Game Option

## 🛠️ Technologies Used
- Python
- Tkinter
- JSON

## 🚀 How to Run

1. Clone this repository
2. Open the project folder
3. Run:

```bash
python snake_game.py
```

## 🎮 Controls
- ⬅️➡️⬆️⬇️ Arrow Keys or WASD - Move Snake
- 🔄 R - Restart Game
